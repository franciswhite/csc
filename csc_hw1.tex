No matter which $x\in X$ wins, agent $i$'s preference order is uniformly (and independently from all other agents) drawn from $m!$ possible such orderings. There are equally many orderings where $x$ occupies the first place, the second place, the third etc. Each place of $x$ in $i$'s order is thus equally probable for this player. Hence, each regret-value - from $m-1$ to $0$ is equally probable; that is, has probabality $\tfrac{1}{m}$. \\
